# 📂 Notebooks Overview

This folder contains all Jupyter notebooks used throughout the project, organized based on development milestones.

---

## 🚩 Milestone 1: Exploratory Data Analysis (EDA)

### 📊 eda_barplots.ipynb
- Performed initial data exploration and visualization  
- Analyzed class distribution using bar plots  
- Verified dataset balance and overall data quality before model training  

---

## 🚀 Milestone 2: Model Development & Evaluation

### 🤖 model_training.ipynb
- Built and trained an image classification model using EfficientNetB0  
- Applied preprocessing techniques:
  - Image resizing  
  - Normalization  
  - Data augmentation  
- Evaluated performance using training, validation, and test datasets  
- Achieved approximately **98% test accuracy** with the optimized model  

---

## 🧪 Experimental Work

### 🔬 haarcascade_experiment.ipynb
- Explored Haar Cascade for preprocessing (region/feature detection)  
- Tested its effect on classification performance  
- Observed inconsistent detection and reduced model accuracy  
- **Conclusion:** Not included in the final pipeline
